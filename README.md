# NeuroBro (PHP build)

Это PHP/HTML/JS сборка сайта (без React).

## Структура
- index.php — лендинг + регистрация/логин
- dashboard.php — личный кабинет
- admin.php — админ-панель
- 404.php — страница 404
- assets/app.js — вся логика на чистом JS + Supabase
- assets/app.css — стили

## Как запускать
Открыть проект через любой PHP-сервер (корень — папка репозитория).

## Примечания
Сайт использует Supabase (auth + БД + RPC).
